# Project: Shangdong Li Portfolio

## Overview
This project is a personal portfolio website for Shangdong Li, designed to be presented to university admissions officers. It showcases a multidisciplinary background in **Technology**, **Music Performance**, and **Music Technology**.

The core philosophy of the portfolio is to demonstrate how technical and musical abilities are applied to solve real-world problems.

## Structure
The project follows a standard static website structure:

*   **`index.html`**: The main entry point. It acts as a Single Page Application (SPA), linking to external CSS and JS resources.
*   **`css/`**: Contains styling files.
    *   `style.css`: Main stylesheet using CSS variables for theming.
*   **`js/`**: Contains JavaScript files.
    *   `script.js`: Handles tab switching logic.
*   **`assets/images/`**: Directory for image assets.
*   **`docs/`**: Reference materials, resumes, and project goals.

## Usage
To view the portfolio:
1.  Open the `index.html` file in any modern web browser (Chrome, Safari, Firefox).

## Development
*   **Modifying Content**: Edit `index.html`.
*   **Styling**: Edit `css/style.css`.
*   **Logic**: Edit `js/script.js`.
*   **Images**: Place images in `assets/images/` and update `<img>` tags in `index.html`.

## Key Themes
1.  **Engineering Solutions**: Deep learning (RAVE), Computer Vision, and Industrial Apps.
2.  **Performance**: Jazz Improvisation (Piano), Classical Foundation, and Orchestral Clarinet.
3.  **Music Tech**: Tech for Social Good (Hearing Accessibility, Alzheimer's Therapy) and Composition.
